🚀 كيفية الاستخدام

على Vercel:
───────────
1. أذهب إلى Project Settings → Environment Variables
2. أضف:
   - MONGODB_URI=...
   - MONGODB_DB_NAME=reputa-analytics
   - PI_API_KEY=...
3. انشر المشروع

على Replit:
───────────
1. أنشئ ملف .env بالمتغيرات
2. شغّل: npm run dev
3. افتح: http://localhost:3000

الاختبار:
────────
curl http://localhost:3000/health
curl http://localhost:3000/api/docs
curl http://localhost:3000/api/admin/dashboard
